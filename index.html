👋 Welcome to Recipe Hub
This viewer loads recipes from a central recipes.json file on SharePoint or OneDrive. Set the link once — every time anyone opens this file it will automatically pull the latest recipes.
1
Upload recipes.json to a SharePoint document library or OneDrive folder.
2
Right-click the file → Share → set to Anyone with the link can view → Copy link.
3
Paste the link below and click Save & Load. This is remembered — staff won't need to do this again.
Works with SharePoint sharing links, OneDrive sharing links, or a direct URL to the raw JSON file.
To update recipes: edit recipes.json and re-upload — everyone gets the new version on next open.
